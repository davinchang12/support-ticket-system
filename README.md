## Run Locally

Clone the project

```bash
  git clone https://github.com/davinchang12/support-ticket-system.git
```

Go to the project directory

```bash
  cd support-ticket-system
```

Install dependencies

```bash
  npm install
```

```bash
  composer install
```

Start the server

```bash
  php artisan migrate --seed
```

```bash
  npm run dev
```

```bash
  php artisan serve
```
